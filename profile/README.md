# [SUT](https://en.sharif.edu) [Mobile Programming](https://docs.ce.sharif.edu/course/40429) Spring 2024 Organization

The Mobile (Mobile Programming) Organization is committed to providing a rich educational resource for students, educators, and enthusiasts in the field of Mobile Programming. Our repositories host a variety of resources, including study materials, practical exercises, and project examples of the course in the SUT (Sharif University of Technology).

## Featured Repository: [Materials](https://github.com/sut-mobile-spring2024/Materials)

The `Materials` repository houses a comprehensive collection of educational materials for Mobile Programming courses taught by Mr. Ahmadi Nik in Spring 2024. It includes assignments, lecture slides, and sources for both students and educators.
